# 2620 凸函数最值

## 题目描述

有一个数学函数F(x)= x7+x6+x3+x2-y*x, 其中y由输入给定, 0&lt;y&lt;100。对于x&gt;=0的范围里，F(x)的最小值是多少？

## 输入格式

输入文件convex.in
输入第一行为正整数T，代表共有多少个问题，T<=10。接着T行，每行一个正整数y。

## 输出格式

输出文件convex.out
输出共T行，每行一个实数，保留3位小数（四舍五入）。

## 输入输出文件

`convex`

## 输入输出样例

输入 #1
```
3
1
2
10
```
输出 #1
```
-0.183
-0.602
-6.474
```
输入 #2
```
无
```
输出 #2
```
无
```
