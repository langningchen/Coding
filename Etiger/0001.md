# 0001 A+B Problem

## 题目描述

输入两个整数a,b，输出它们的和.&nbsp;题目保证输入数据的范围如下：-1000000000&lt;=a&lt;=1000000000-1000000000&lt;=b&lt;=1000000000

## 输入格式

输入两个整数，以空格分开

## 输出格式

输出一个整数

## 输入输出样例

输入 #1
```
20 30
```
输出 #1
```
50
```
输入 #2
```
6 9
```
输出 #2
```
15
```
