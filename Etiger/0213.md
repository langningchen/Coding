# 0213 反向输出

## 题目描述

输入n个字符(1<=n<=100)，反向输出这些字符，由空格隔开。

## 输入格式

输入格式：第一行是一个整数n，代表共有多少字符。之后一行有n个字符由空格隔开。

## 输出格式

n个数，空格隔开
