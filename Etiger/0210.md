# 0210 阶乘

## 题目描述

输入一个正整数n,输出n的阶乘，也就是n*(n-1)*(n-2)*…*2*1连乘的结果。例如5的阶乘是5*4*3*2*1=120。&nbsp;&nbsp;注意：int类型范围不够大时可以考虑long long类型

## 输入格式

一个整数n，0<=n<=18

## 输出格式

一个整数

## 输入输出样例

输入 #1
```
6
```
输出 #1
```
720
```
输入 #2
```
0
```
输出 #2
```
1
```
