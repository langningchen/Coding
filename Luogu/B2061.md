# B2061 整数的个数
***您已经通过此题！***

## 题目描述

给定 $k$（$1<k<100$）个正整数，其中每个数都是大于等于 $1$，小于等于 $10$ 的数。写程序计算给定的 $k$ 个正整数中，$1$，$5$ 和 $10$ 出现的次数。

## 输入格式

输入有两行：第一行包含一个正整数 $k$，第二行包含 $k$ 个正整数，每两个正整数用一个空格分开。

## 输出格式

输出有三行，第一行为 $1$ 出现的次数，，第二行为 $5$ 出现的次数，第三行为 $10$ 出现的次数。

## 输入输出样例

输入 #1
```
5
1 5 8 10 5 
```
输出 #1
```
1
2
1
```

## 最后一次提交代码

```
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
ll k, ans1, ans2, ans3;
int main()
{
    cin >> k;
    for (int i = 0; i < k; i++)
    {
        ll t;
        cin >> t;
        if (t == 1)
            ans1++;
        if (t == 5)
            ans2++;
        if (t == 10)
            ans3++;
    }
    cout << ans1 << endl
         << ans2 << endl
         << ans3 << endl;
    return 0;
}
```
