# B2071 余数相同问题
## 题目描述

已知三个正整数 $a$，$b$，$c$。现有一个大于 $1$ 的整数 $x$，将其作为除数分别除 $a$，$b$，$c$，得到的余数相同。

请问满足上述条件的 $x$ 的最小值是多少？数据保证 $x$ 有解。

## 输入格式

一行，三个不大于 $1000000$ 的正整数 $a$，$b$，$c$，两个整数之间用一个空格隔开。

## 输出格式

一个整数，即满足条件的 $x$ 的最小值。

## 输入输出样例

输入 #1
```
300 262 205
```
输出 #1
```
19
```
