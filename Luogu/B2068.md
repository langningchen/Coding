# B2068 统计满足条件的 4 位数
## 题目描述

给定若干个四位数，求出其中满足以下条件的数的个数：个位数上的数字减去千位数上的数字，再减去百位数上的数字，再减去十位数上的数字的结果大于零。

## 输入格式

输入为两行，第一行为四位数的个数 $n$，第二行为 $n$ 个的四位数。$(n \le 100)$。

## 输出格式

输出为一行，包含一个整数，表示满足条件的四位数的个数。

## 输入输出样例

输入 #1
```
5
1234 1349 6119 2123 5017
```
输出 #1
```
3
```
