# B2046 "骑车与走路"
## 题目描述

在清华校园里，没有自行车，上课办事会很不方便。但实际上。并非去办任何事情都是骑车快，因为骑车总要找车、开锁、停车、锁车等，这要耽误一些时间。假设找到自行车，开锁并车上自行车的时间为 $27$ 秒；停车锁车的时间为 $23$ 秒；步行每秒行走 $1.2$ 米，骑车每秒行走 $3.0$ 米。请判断走不同的距离去办事，是骑车快还是走路快。

## 输入格式

输入一行，包含一个整数，表示一次办事要行走的距离，单位为米。

## 输出格式

- 如果骑车快，输出一行 `Bike`；

- 如果走路快，输出一行 `Walk`；

- 如果一样快，输出一行 `All`。

## 输入输出样例

输入 #1
```
1
```
输出 #1
```
Walk
```
