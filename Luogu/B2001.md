## 题目描述

求两个整数的和。

## 输入格式

一行，两个用空格隔开的整数。

## 输出格式

两个整数的和。

## 输入输出样例

输入 #1
```
1 2
```
输出 #1
```
3
```
输入 #2
```
10230 21312
```
输出 #2
```
31542
```

## 提示

对于 $100\%$ 的数据，输入的整数在 $[1, 4 \times {10}^{18}]$ 内。
